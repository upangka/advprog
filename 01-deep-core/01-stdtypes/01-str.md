

# 常用API

##  str.isspace

[str.isspace](https://docs.python.org/3/library/stdtypes.html#str.isspace)

```pytho
>>> ''.isspace() # 空字符串
False
>>> ' '.isspace()
True
>>> '\t\n'.isspace() # TAB and BREAK LINE
True
>>> '\u3000'.isspace() # IDEOGRAPHIC SPACE
True
```

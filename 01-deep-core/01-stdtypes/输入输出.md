# 重定向输出输出

运行含有`input`的程序，可以在运行程序的时候，可以指定输入文件，同时需要输出的`print`语句，也可以在运行程序的时候指定输出文件

```sh
# < 输入文件  > 输出文件
uv run exercise_01.py < exercise_01_input.txt > exercise_01_out.txt
```

[exercise_01.py](./code/input_ouput/exercise_01.py)

```python
infos = [input().upper() for _ in range(2)]
print(",".join(infos))
```

运行

```sh
# 此时只有两个文件，并没有输出文件
$ ls
exercise_01.py  exercise_01_input.txt

# 输入文件的内容
$ cat exercise_01_input.txt
hello
world

# 运行程序指定输入和输出文件
$ uv run exercise_01.py < exercise_01_input.txt > exercise_01_out.txt

# 查看生成的输出文件
$ cat exercise_01_out.txt
HELLO,WORLD
```

```python
>>> hypot = [("push", 2), ("pow",), ("swap",), ("push", 2), ("pow",), (\
"add",), ("sqrt",)]
>>> 
>>> for a,*b in hypot:
...     print(f"{a=} {b=}")
...
a='push' b=[2]
a='pow' b=[]
a='swap' b=[]
a='push' b=[2]
a='pow' b=[]
a='add' b=[]
a='sqrt' b=[]
```

空参数函数的调用方式

```python
>>> def func():
...     print("hello world")
...
>>> func()
hello world
>>> func(*[])
hello world
>>> func(**{})
hello world
```

func(*[]) 和 func(**{}) 之所以能工作，是因为：
1. `func()` 本身不接收任何参数
2. `*[]` 解包出一个空的位置参数列表 → 等同于没有传参
3. `**{}` 解包出一个空的关键字参数字典 → 等同于没有传参

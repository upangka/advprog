
```python
>>> class A:
...     def __init__(self,x):
...         self._x = x
...         self.__x = x
...
...     def _f1(self):
...         pass
...
...     def __f1(self):
...         pass
...
>>> # 查看属性
>>> a = A(3)
>>> vars(a)
{'_x': 3, '_A__x': 3}
>>> a._x
3
>>> a._A__x
3
>>> # 查看方法
>>> [item for item in dir(a) if 'f1' in item]
['_A__f1', '_f1']
>>> a._f1()
>>> a._A__f1()
```


| 特性 | `vars()` | `dir()` |
|------|----------|---------|
| 返回内容 | 实例的属性字典（只包含实例变量） | 对象的所有属性和方法名称列表（字符串列表） |
| 包含属性 | ✅ 实例变量（如 `_x`, `_A__x`） | ✅ 所有属性（包括实例变量） |
| 包含方法 | ❌ 不包含方法 | ✅ 所有方法（包括私有方法） |
| 包含继承的 | ❌ 仅当前实例 | ✅ 包含父类继承的属性和方法 |
| 返回类型 | `dict`（键值对） | `list`（字符串列表） |
| 是否显示私有/保护 | ✅ 显示（`_x`, `_A__x`） | ✅ 显示（`_f1`, `_A__f1`） |
| 名称修饰 | ✅ 显示修饰后的名称（`_A__x`） | ✅ 显示修饰后的名称（`_A__f1`） |
